The example of my code

A JavaScript function which takes three different arrays and returns an array. This array includes non-repeatable values without zeros. And all values have been sorted. 

          function makeSet(array1, array2, array3) {
            return Array.from(new Set([...array1, ...array2, ...array3]))
              .filter(num => num !== 0)
              .sort((a, b) => a - b);
          }

          const arr1 = [8, 3, 2, 8, 0, 4];
          const arr2 = [1, 4, 0, 2, 5];
          const arr3 = [9, 0, 5, 0];

          makeSet(arr1, arr2, arr3); //result - [1, 2, 3, 4, 5, 8, 9]
